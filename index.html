Discord Bot (Demo)

/check-user (discordId)

Dieser Bot ist eine Demo-Version für Kunden: Er simuliert einen Check eines Discord-Users gegen interne Risiko-Signale (z.B. bekannte Cheat-Communities, Re-Join Pattern, Link-Cluster). In einer echten Version werden die Datenquellen/Listen/Regeln individuell für dein Netzwerk angebunden.

/check-user Demo ID

Was der Bot kann

01

Slash Command: /check-user <discordId> mit strukturierter Ausgabe (Risk Score, Flags, Hinweise).

02

Server-/Community-Risiko-Matching (Demo: simuliert). Real: Matching über gepflegte Watchlists + Heuristiken.

03

Audit-Logs & Export (z.B. JSON/Webhook). Real: Integrationen in Admin-Panel/CRM möglich.

Demo Bot Code (Node.js)

// Demo: discord.js v14 (vereinfachtes Beispiel) // /check-user <discordid> - liefert Risiko-Signale (simuliert) import { Client, GatewayIntentBits, SlashCommandBuilder } from "discord.js"; const client = new Client({ intents: [GatewayIntentBits.Guilds] }); const command = new SlashCommandBuilder() .setName("check-user") .setDescription("Check a Discord user against risk signals") .addStringOption(opt => opt.setName("discordid").setDescription("Discord ID").setRequired(true)); function simulateCheck(discordId) { const flagged = Math.random() < 0.35; const score = flagged ? 70 + Math.floor(Math.random() * 30) : 5 + Math.floor(Math.random() * 35); const flags = flagged ? ["watchlist_match", "invite_cluster", "alt_pattern"] : ["no_match"]; return { discordId, score, flags }; } client.on("interactionCreate", async (i) => { if (!i.isChatInputCommand() || i.commandName !== "check-user") return; const discordId = i.options.getString("discordid", true); const result = simulateCheck(discordId); await i.reply({ ephemeral: true, content: `Result for ${result.discordId}: Risk=${result.score}/100 Flags=${result.flags.join(", ")}`, }); }); client.login(process.env.DISCORD_TOKEN);

Hinweis: Diese Website-Demo zeigt nur eine Simulation. Für eine echte Umsetzung müssen Datenquellen, Datenschutz und zulässige Prüf-Logik definiert werden (keine „Random“-Checks).
